    Alert

    An alert is a dialog that presents users with information, alert messages. An alert appears on top of the web app's content and must be manually dismissed before user can resume interacting with the app.

    Demo

    Alert Heading!
    Alert text description.

    HTML

                  
<div class="container alert" id="alert-container">
<div class="container-title">Alert Heading!</div>
<div class="container-content">Alert text description.</div>
<div class="container-footer">
<button class="btn btn-full-width" onclick="closeAlert()">Dismiss</button>
</div></div>

    JS

      
const showAlert = () => document.getElementById("alert-container").style.display="unset";
const closeAlert = () => document.getElementById("alert-container").style.display = "none";

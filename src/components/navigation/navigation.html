    Navigation

    The navigation component is built on flexbox to provide a strong foundation for building all types of navigation components. It contains left/right list items and icons/text tabs.

    • Tab Active

<ul class="nav-group start">
<li> <a href="#">Tab 1</a></li>
<li class="nav-active"><a href="#">Tab Active</a></li>
</ul>
<ul class="nav-group end">
<li><a href="#"><i class="icon icon-md fa fa-github"></i></a></li>
<li><a href="#"><i class="icon icon-md fa fa-github"></i></a></li>
<li></li>
</ul>
</nav>
